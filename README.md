# Sports Scoreboard Console Application
Simple console application that simulates a sports scoreboard system. The application allows users to manage live games by adding matches, updating scores, finishing games, and displaying a summary of all matches.


Example scoreboard:

id: 1. Mexico - Canada: 5-0  
id: 2. Spain - Brazil: 3-2  
id: 3. Germany - France: 2-3

Technologies

- Java
- Console input/output
